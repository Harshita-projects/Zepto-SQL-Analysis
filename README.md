# Zepto Inventory & Sales Analysis (SQL)

# 📊 Project Overview
This project is an end-to-end SQL analysis of Zepto's grocery inventory. As a B.Com graduate and aspiring Analyst, I used this project to uncover business insights regarding revenue, stock levels, and logistics optimization.

# 🛠️ Technical Skills Applied
* Data Cleaning:Handled NULL values and removed incorrect ₹0 MRP entries to ensure data integrity.
* Financial Metrics:Calculated estimated revenue per category using mathematical operations in SQL.
* Advanced Logic:Created a logistics classification system (Low, Medium, Bulk) using `CASE` statements.
* Data Filtering:Identified high-value "revenue leaks" by finding expensive items that are out of stock.

# 💡 Key Business Insights
* Top Revenue Categories:Identified which departments contribute most to the total inventory value.
* Discount Strategy:Analyzed the top 10 best-value products based on discount percentage.
* Stock Management:Pinpointed premium products (MRP > ₹500) that need urgent restocking.

# 📂 Repository Contents
* `zepto_analysis_project.sql`: Full MySQL script with all analysis queries.
* Screenshots: Visual results of the most critical data cleaning and analysis steps.
